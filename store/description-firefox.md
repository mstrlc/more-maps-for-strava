Enhance Strava's route builder and heatmap with map layers from Mapy.cz, OpenStreetMap, and Google Maps. Click anywhere on the map to view street-level panoramas from Mapy.cz or Google Street View. Customize map layer opacity and saturation to your liking.


🗺️ **MAP LAYERS**

- **Mapy.cz** — Standard, Outdoor, Winter, Satellite
- **OpenStreetMap** — Standard, CyclOSM, Cycle Map (Thunderforest)
- **Google Maps** — Standard, Satellite, Terrain, Hybrid


👁️ **PANORAMA MODE**

Click anywhere on the map to open a draggable, resizable panorama window with street-level imagery. Switch between Mapy.cz and Google Street View using the control in the panorama window. Press **P** to toggle panorama mode on/off.


🎨 **MAP STYLING**

Adjust the opacity and saturation of any custom map layer — from full color to grayscale, fully opaque to transparent.


⚙️ **SETUP**

Some features require a free API key. Click the _More Maps Settings_ button in Strava's map layer menu to enter your keys. The extension will prompt you automatically the first time you use a feature that needs one. No credit card required. Free tiers cover all personal use.

🔑 Keys are stored exclusively in your browser's local storage and never leave your device.


**Mapy.cz (~1 min)** — required for Mapy.cz map layers and Mapy.cz Panorama

1. Go to [developer.mapy.com/account/projects](https://developer.mapy.com/account/projects)
2. Create a free account and a new project
3. Copy the API key and paste it into the extension settings


**Google Maps (~3 min)** — required for Google map layers and Google Street View

1. Go to [console.cloud.google.com/google/maps-apis/credentials](https://console.cloud.google.com/google/maps-apis/credentials)
2. Create a free account and a new project
3. Enable the **Map Tiles API** and the **Street View Static API** for your project
4. Create an API key, copy it, and paste it into the extension settings


**Thunderforest (~1 min)** — required for the Cycle Map layer

1. Go to [manage.thunderforest.com](https://manage.thunderforest.com/)
2. Create a free account
3. Copy the API key and paste it into the extension settings
